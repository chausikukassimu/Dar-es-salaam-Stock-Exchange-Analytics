This repository contains a comprehensive collection of financial analyses, quantitative research, and data-driven insights related to the Dar es Salaam Stock Exchange (DSE). It serves as a central hub for exploring the performance and behavior of Tanzanian listed equities through the application of financial theory, statistical methods, and modern data analytics techniques. The repository includes stock performance analysis, portfolio construction and optimization, risk and return measurement, volatility modeling, market efficiency studies, liquidity and spillover analyses, network-based market investigations, and interactive data visualizations. Built primarily with Python, the project aims to support investors, researchers, students, and financial analysts in understanding the dynamics of the Tanzanian capital market and promoting evidence-based investment decision-making through reproducible and transparent analytical workflows.
